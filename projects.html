OVER8TEEN

CONTEXT: SIDE PROJECT

YEAR: 2022

Small side project I did for my colleagues at my part-time job as a bartender. The solution was made to stop a repeating pattern after working 6 years within the industry

PROCESS

01

FEATURES, TECH AND TOOLS
- LIGHT/DARK THEME
- WORKING WITH DATES
- LOCAL STORAGE
- HTML, CSS, JAVASCRIPT

02

BACKGROUND
I saw a tendency that a lot of my colleagues had a hard time counting backwards on when people were old enough to be served alcohol. I have seen over sites during this, but never with the specific date and an ability for dark mode.

03

RESULT
Small project which calculates the date people were born to be 18 years old. I have seen website during it with the year before, but never with the specific date and an ability for dark theme website with local storage.

WEB CARBON CALCULATOR

CONTEXT: SCHOOL PROJECT

YEAR: 2022

This school group project was created in collaboration with the company Advice. We were giving a case to build a website including a form that could calculate how dirty your website is regarding the environment.

PROCESS

01

FEATURES, TECH AND TOOLS
- HTML, CSS AND JS
- REST API
- FORM
- NETLIFY
- BRANCHES
- GIT

02

BACKGROUND
The focus was to re-think the carbon calculator that already exists working with user experience and create curiosity and awareness for people using the website. How to present all this information smooth, make it interactive and get data from our own database.

03

RESULT
A web carbon calculator website, which includes a smooth form, design that optimizes user experience and creates awareness.

Next step: would be to implement a field in the result page, where you can check out other companies to compare.
